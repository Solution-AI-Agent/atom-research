Key Trends
핵심 트렌드 분석
2025년 LLM 시장을 정의하는 5가지 핵심 트렌드.
1
추론 모델의 부상
DeepSeek R1(1월)이 MIT 라이선스로 추론 모델 시대를 열었고, o3(4월), GPT-5 thinking(8월), Grok 4.1(11월)까지 이어지며 "생각하는 AI"가 새로운 표준이 됐습니다. 핵심 기술은 RLVR(Reinforcement Learning with Verifiable Rewards) 기반 추론으로, 수학/과학에서 기존 대비 20~40pp 향상. 특히 코딩 디버깅에서 탁월 -- SWE-bench에서 추론 모델이 비추론 모델 대비 평균 15pp 이상 높은 성능을 보였습니다.
DeepSeek R1
o3 / o4-mini
Grok 4.1

2
에이전트 AI 폭발
Gartner 기준 에이전트 AI 관련 문의가 1,445% 증가. 시장 규모는 $7.8B(2025) -> $52B(2030) 전망. Claude Opus 4의 SWE-bench 72.7%, GPT-5의 멀티모달 에이전트, Grok 4의 자율 코딩 등 모든 주요 벤더가 에이전트 능력을 핵심 차별점으로 내세우고 있습니다. MCP(Model Context Protocol)가 AAIF(Linux Foundation)에 기부되면서 에이전트 간 상호운용성의 공식 표준이 확립됐습니다.
Gartner 1,445%

MCP + AAIF
3
MCP 표준화: 프로토콜에서 인프라로
2024.11 Anthropic이 MCP 발표 -> 2025.03 OpenAI가 채택 -> 2025.05 업계 표준화 -> 2025.12 Linux Foundation(AAIF)에 기부. npm 다운로드 기준 100K -> 800만 (80배 성장). 에이전트가 외부 도구/데이터소스에 접근하는 방식이 표준화되면서, LLM 에코시스템의 "USB-C"로 자리잡았습니다. 모든 주요 IDE(VS Code, JetBrains, Cursor)와 클라우드(AWS, Azure, GCP)가 MCP를 지원합니다.
2024.11 Anthropic 발표
2025.03 OpenAI 채택
2025.12 Linux Foundation 기부
다운로드 80x
4
오픈소스 vs 프로프라이어터리
오픈소스/오픈웨이트 모델이 전체 LLM 사용의 62.8%를 점유. DeepSeek-V3(MIT), Mistral Large 3(Apache 2.0), Llama 4(Community License) 등이 주도. 하지만 최고 성능은 여전히 프로프라이어터리 -- SWE-bench 상위 5개 중 4개가 폐쇄형(Claude Opus 4, o3, GPT-5, Grok 4.1). 실무적으로는 "오픈소스로 80% 커버, 나머지 20%는 API"라는 하이브리드 전략이 확산 중입니다.
오픈소스 62.8% 점유
Top 성능은 Proprietary
하이브리드 전략 확산
5
가격 붕괴: GPT-4 수준이 1달러 미만
2023년 GPT-4 수준 성능의 API 비용: $30/1M tokens -> 2025년 $1 미만 (30배 이상 하락). Gemini 2.5 Flash($0.15/$0.60), DeepSeek-V3($0.27/$1.10)가 GPT-4를 능가하면서 초저가. 이 가격 붕괴로 "Good Enough" 모델 + 대량 처리 패턴이 주류가 됐고, Frontier 모델은 고난도 추론/에이전트에만 사용하는 티어링 전략이 보편화됐습니다.

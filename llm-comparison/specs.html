LM Arena (LMSYS) 순위 참고

Insights
핵심 분석
1. 코딩/SWE-bench: Anthropic Claude 4.6 압도적 우위

Claude Opus 4.6 (80.8%)과 Sonnet 4.6 (79.6%)이 SWE-bench Verified에서 모든 경쟁 모델을 큰 격차로 앞서고 있습니다. o3 (69.1%)이 3위이며, Gemini 2.5 Pro (63.8%)가 뒤를 잇습니다.

2. 수학/추론: o4-mini와 DeepSeek-R1의 전문성

o4-mini는 AIME 2025에서 92.7% (도구 없이)를 달성하며 수학에서 가장 뛰어난 모델입니다. DeepSeek-R1은 MATH-500에서 97.3%를 기록. Grok 3 Think는 consensus@64에서 93.3%를 보여줍니다.

3. 과학 추론 (GPQA): Claude Opus 4.6 선두

GPQA Diamond에서 Claude Opus 4.6 (91.3%)이 압도적 1위. Grok 3 Think (84.6%), Gemini 2.5 Pro (84.0%), o3 (82.8%)이 80% 이상 그룹을 형성합니다.

4. 컨텍스트 윈도우: 1M 토큰 시대

GPT-4.1, Claude 4.6 시리즈, Gemini 2.5 시리즈, Llama 4 Maverick, Grok 3 모두 1M 토큰을 지원합니다. Llama 4 Scout는 10M 토큰으로 최대 컨텍스트 윈도우를 보유합니다.

5. 오픈소스: MoE 아키텍처가 표준

Llama 4 (400B/17B), Mistral Large 3 (675B/41B), DeepSeek-V3/R1 (671B/37B), Qwen3 (235B/22B) 모두 MoE 구조를 채택. 활성 파라미터 대비 총 파라미터 비율로 성능과 효율을 동시에 달성합니다.

6. 멀티모달: GPT-4o가 가장 범용적

GPT-4o는 텍스트+비전+오디오를 단일 모델로 처리하는 유일한 모델. Gemini 2.5 시리즈도 텍스트+비전+오디오+비디오를 지원합니다. DeepSeek-V3/R1은 텍스트 전용으로 멀티모달 미지원 (별도의 VL2/Janus 모델 필요).

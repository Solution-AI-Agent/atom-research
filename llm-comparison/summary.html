Deployment
배포 의사결정 매트릭스
Cloud API
  • 즉시 사용 가능, 인프라 투자 불필요
  • SOC2/HIPAA/GDPR 인증 완비 (OpenAI, Anthropic, Google)
  • 자동 스케일링, 99.9% SLA
  • 최신 모델 즉시 접근
Cloud API 단점
  • 대량 사용 시 비용 급증 (o3: $40/1M output)
  • 데이터가 외부 서버를 경유
  • 벤더 종속, 가격 변동 리스크
  • 파인튜닝 옵션 제한 (Claude: FT 미지원)
Self-hosted
  • 완전한 데이터 주권, 규제 환경 적합
  • 대량 처리 시 한계 비용 감소
  • 자유로운 파인튜닝 (LoRA/QLoRA)
  • MIT/Apache 2.0 라이선스 = 제한 없음
Self-hosted 단점
  • H100 GPU 필요 (Scout 1대, V3/R1 8대 이상)
  • 초기 인프라 비용 및 운영 인력 필요
  • 모델 업데이트를 직접 관리
  • SLA/모니터링 자체 구축 필요
Risk Advisory
핵심 리스크
DeepSeek API 데이터 프라이버시
중국 서버 저장: 모든 데이터가 PRC 서버에 저장, 중국 국가정보법에 따라 정부 접근 가능.
SOC2/HIPAA 미인증: Zero Data Retention 미제공. 데이터 보존 기간 "가능한 오래" 명시.
보안 사건: 2025년 1월 공개 DB에 100만 건 이상 채팅 기록/API 키 노출.
GDPR 조사 중: 이탈리아, 아일랜드, 벨기에 등 5개국에서 조사 진행. 한국 개인정보위 무단 해외 이전 확인.
권장: DeepSeek 모델 사용이 필요한 경우 반드시 셀프호스팅(MIT 라이선스)으로 데이터 유출 리스크 해소.

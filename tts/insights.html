EL
ElevenLabs
가격: $206/1M chars — ElevenLabs 공식 가격표 (Scale 플랜 기준)
모델: Multilingual v2 (감정), Flash v2.5 (저지연) — 동일 가격
레이턴시: Flash v2.5 ~75ms — 공식 벤치마크
스트리밍: WebSocket 실시간 — 공식 API 문서
OA
OpenAI
가격: ~$15/1M chars (환산) — OpenAI TTS API 공식 가격 ($12/1M audio tokens)
모델: gpt-4o-mini-tts — 감정/스타일 제어 가능
보이스: 6개 범용 보이스 (한국어 전용 없음) — 공식 문서
Rate Limit: Tier 1: 7 RPM — 공식 Rate Limits 페이지
GC
Google Cloud
Neural2: $16/1M chars — Google Cloud TTS 공식 가격표
무료 티어: 1M chars/월 (Neural2) — 공식 무료 티어
Journey: $30/1M chars — 대화형 고급 보이스
SLA: 99.9% — Google Cloud SLA 문서
AZ
Azure
Neural: $15/1M chars — Microsoft Azure Speech 공식 가격표
무료 티어: 0.5M chars/월 — 공식 무료 티어
한국어 보이스: 12개 Neural + DragonHD 2개 — Azure Speech 보이스 갤러리
한자 카운트: 한자 1글자 = 2글자 카운트 — Microsoft 공식 문서 확인
산출 방식: TCO 시뮬레이터는 월간 문자 생성량에 각 서비스의 공식 단가를 적용하고, 무료 티어가 있는 서비스는 무료 할당량을 차감한 후 계산합니다. Google Cloud는 월 1M chars, Azure는 월 0.5M chars 무료입니다. 실제 비용은 보이스 모델 선택, 스트리밍 여부, 한자 비율에 따라 달라질 수 있습니다.
